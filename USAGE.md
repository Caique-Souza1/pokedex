# Guia de uso

## Página inicial

Na página principal é possível:

- Selecionar a geração desejada
- Buscar Pokémon pelo nome
- Acessar a página de detalhes
- Favoritar Pokémon clicando no ícone de coração

A lista é atualizada automaticamente ao trocar a geração.

## Página de detalhes

Ao clicar em "Detalhes", são exibidas informações como:

- Nome
- Arte oficial
- Altura
- Peso
- Tipo
- Habilidade principal

## Página de favoritos

Mostra todos os Pokémon adicionados como favoritos.

Os favoritos ficam salvos no navegador utilizando LocalStorage, então mesmo fechando a página os dados continuam armazenados.

É possível remover dos favoritos clicando novamente no coração.
